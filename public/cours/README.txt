COURS INTERACTIF — Révision soutenance Laravel
==============================================

Deux façons de l'ouvrir :

1) Le plus simple (sans rien lancer) :
   - Double-clique sur le fichier  index.html  (il s'ouvre dans ton navigateur).

2) Via l'application Laravel (quand le serveur tourne) :
   - Avec Laravel Sail (Docker, ce projet) : http://localhost:8080/cours
   - Avec "php artisan serve" : http://localhost:8000/cours

Comment l'utiliser :
   - En haut : choisis une FONCTIONNALITÉ (Créer un article, Passer une commande,
     Se connecter, Ajouter au panier).
   - À gauche : l'ARBORESCENCE du projet (quel fichier, dans quel dossier).
     Clique un fichier pour l'ouvrir.
   - Au centre : le CODE (coloré). À droite : l'EXPLICATION ligne par ligne.
     Survole une ligne de code -> son explication s'illumine.
   - Onglets : "Schéma MVC" (le parcours), "Quiz du jury", "Glossaire".
   - Flèches clavier ← → : étape précédente / suivante.
   - Boutons : 🌙 thème clair/sombre, 🖨️ imprimer en PDF.

100 % local — aucune connexion Internet requise.
